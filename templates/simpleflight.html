A simple scene with simulated gliding physics, written in C using OpenGL/GLUT

Written as an open-ended final project for a graphics class, this was intended as a small demonstration of dynamic lighting and manual object modelling, but I'm also proud of the linear algebra functions on lines 141 to 247 of "sf.c". These functions serve both to do the math required for keeping track of the location and heading of the airplane, and to apply the modelled physical forces on the plane ("physics_update()" on 1774-1833) in the correct directions, and which in my opinion express the underlying math very clearly.

The github repo can be found here, and there is a link to directly download here.

To use:
* Unzip simpleFlight.zip into any directory
* From the terminal, navigate to that directory and type (ignoring '$')
$ make
$ ./sf
or just
$ ./run.sh
* Move the plane with the mouse, and tap space once or twice if you need a "push" to climb a little higher! Additional controls are in the README.
